Privacy

This site doesn't collect any information, cookies, or store any analytics. The priority is and always will be on maintaining user safety.

What the site comprises of

The whole site is currently just static HTML, CSS, and JS files.

There's currently no backend, server-side code, or user data anywhere on this public repo. There are plans to turn this into a fully functioning website with my own security measures implemented, at which point I will update this description and any applicable guidelines too.

Outbound links

If I link to an external site, that site would have its own privacy practices. Due to that, I implemented a no-referrer policy so that outbound clicks don't reveal the destination where you came from.
